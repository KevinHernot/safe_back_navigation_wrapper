# Code of Conduct

This project aims to be welcoming, respectful, and collaborative.

## Expected Behavior

- be kind and constructive
- assume good intent
- focus feedback on ideas and code, not people
- help keep the project accessible to learners

## Unacceptable Behavior

- harassment or personal attacks
- discriminatory language
- repeated bad-faith disruption

Project maintainers may moderate discussions and contributions to keep the space healthy.
